## 项目背景

随着AI技术的发展，我开始探索如何将AI能力集成到日常开发工作中。这个项目是我的第一个尝试，旨在创建一个能够理解代码上下文的AI助手。

## 技术栈

- **前端**: Next.js 14 App Router
- **AI**: Claude API
- **数据库**: Prisma + PostgreSQL
- **认证**: NextAuth.js

## 核心功能

1. 代码补全与优化建议
2. Bug分析与修复方案
3. 项目文档自动生成
4. 代码重构建议
